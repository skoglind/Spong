# Spong
A Pong clone.

    JDK: Java 1.8
    Frameworks: AWT and Swing

### Installation
Compile alla java-files, then run the Game-class.

    javac *.java
    java Game
    
#### Game instructions
How to play the game?

    Player 1,
        Arrow Up
        Arrow Down
        
    Player 2,
        1
        Q
        
    P to pause game
    ESC to exit pause screen, game and game
